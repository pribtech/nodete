Credits for the Technology Explorer for IBM DB2

Who helped to build the Technology Explorer for IBM DB2?

The Technology Explorer for IBM DB2 was designed and built off of the DB2 Monitoring Console written by Peter Kohlmann and Matthew Vandenbussche with significant help from many other people.

The current Technology Explorer team:

  • Clara Liu - Project manager and domain expert
  • Paul Johnson - Lead developer, release manager, test architect
  • Carmen Wong - Demonstration developer
  • Peter Prib - DBA interface developer and consultant
  • Andrew Heard - Developer and QA
  • Erik Grandy - Developer and QA
  • Gord Davison - Media design consultant
