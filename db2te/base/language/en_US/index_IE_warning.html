Warning you are using Internet Explorer!

Although we support Internet Explorer it is highly recommended that you use one of the following browsers.

  • Chrome
  • Firefox 3+
  • Safari 3+

As an internet application the Technology Explorer uses a lot of JavaScript. Specifically with the new table rendering engine. Larger views such as the page View->Catalog->Tables, will take significantly longer to load in IE (two to ten times as long as Firefox 3). Considerable time has been spent optimizing the table rendering engine for IE and against the architects philosophy a separate table rendering engine has been created specifically for IE. This engine removes some nice to have features available in other browser, significantly improving the rendering time in IE.

The modified rendering engine can be disable by setting the configuration variable IE_SPEED_EXTENSION to the boolean value false in the config.php file.

This message can be disabled by setting the configuration variable SHOW_IE_PERFORMANCE_WARNING to the boolean value false in the config.php file.
