Welcome to the Technology Explorer for IBM DB2 base start page

This page should shortly be replaced shortly when the web application has completely loaded.

If this page remains then you likely are having one of these three problems:

PHP problems:

The TE requires PHP 5.2 but can run in PHP 5.1 given the following modules are available:

  • IBM_DB2 – Will not stop the TE from loading but will prevent you from connecting
  • JSON
  • DOM XML (Use of native or simple XML is supported but requires a configuration change)

Note: On some Linux distribution the native PHP instillation disables JSON for the web server but not the CLI.

JavaScript problems:

  • Your browser does not have JavaScript support
  • IE has disabled JavaScript support
  • You have turned off JavaScript with in your browser.

As The Technology Explorer for IBM DB2 is a dynamic web application it requires JavaScript to function. We have tested the TE in the following browsers:

  • IE 6 *
  • IE 7 *
  • IE 8 *
  • Opera 9+
  • Safari 3+
  • Firefox 3+
  • Chrome 2+

* Not recommended due to JavaScript performance

Aggressive browser caching

Most modern browser cache aggressively in most cases this is a good thing but can cause a fixed error to appear to persist. Clear your cache and try loading the page again.

If you are still having trouble post on the forums with you system and setup information and someone will try to help you.
