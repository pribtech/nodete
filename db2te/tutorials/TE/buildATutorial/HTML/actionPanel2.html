To the left is the framework for a task. Tasks contain 3 major components:

  1. 'name' attribute
    • this is the label that will appear on the button to run this task
  2. 'description' element
    • A displayed description of the action that will be displayed when the user hovers over the button
  3. 'action' element
    • contains the XML action script to run

Steps to complete:

  • Copy the contents of tab 'joinTiming.xml' located to the right into the file 'joinTiming.xml'.
  • Save the file, but do not close it
  • Reload the 'About table joins' tutorial navigate to the last page. An action panel with one button should now be displayed on the right hand side.
