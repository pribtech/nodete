Now that you have a saved XML table definition for the Dependencies page, you should switch the target of the reference in the Tables page from the system catalog table to the XML table definition file.

Goal:

Switch the reference in the table view to the XML table definition file we just saved. You can use the modified file displayed to the right as an example.

Steps to complete:

  • If you closed the tablecat.xml file
    • Open a terminal and run the following command:
      • “gedit /var/www/tableDefinitions/tablecat.xml &”
  • Go to the Dependencies reference
  • Edit the value of the element 'refvalue' to 'tableDependCat' (an extension of lowercase '.xml' is implied)
