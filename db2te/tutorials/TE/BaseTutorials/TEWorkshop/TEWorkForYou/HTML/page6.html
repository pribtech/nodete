The updated Table page is reloaded in the right window.

You should see a new column labeled "Dependencies" in the page.

Goal:

Check and verify that the new Dependencies column is working.

Steps to complete:

Find a new column named 'Dependencies'. Click on the 'Dependencies' column in either 'DEMOADM.EMP' or 'DEMOADM.DEPT'. The dependencies are listed in the lower panel.

Troubleshooting:

You may need to right click on the browser, and reload TE. If you see a blank page to the right, it is possible that you have made an error with your editing. Open a new tab in the browser and navigate to the table definition file (http://localhost/tableDefinitions/tablecat.xml). If there is an XML parsing error Firefox should highlight the problem for you. Otherwise, go back and review your changes.
