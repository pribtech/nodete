Descriptions of edits being made:

  • Column titles will be changed to make them more readable
  • Presentation type will be set to appropriately display the column data
    • Presentation types are:
      • 'n' for numbers
      • 's' for strings
      • 'l' for large blobs of text.
  • Sorting will be explicitly enabled on all columns by the addition of the 'c_sort' element containing the value 'y' to indicate that the column is sortable
