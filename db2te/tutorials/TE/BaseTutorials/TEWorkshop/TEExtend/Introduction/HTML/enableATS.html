The DB2 LUW Administrative Task Scheduler is disabled by default. Even though the Administrative Task Scheduler is disabled, users with proper authorities can create, modify, and remove tasks as well as view existing tasks. However, to execute tasks the Administrative Task Scheduler must be enabled.

To enable the DB2 LUW Administrative Task Scheduler you must set the DB2 registry variable DB2_ATS_ENABLE to one of the following 4 values: YES, TRUE, 1, or ON.

The command to set the registry variable is displayed on the right. The database instance must be stopped and started before the change will take effect.
