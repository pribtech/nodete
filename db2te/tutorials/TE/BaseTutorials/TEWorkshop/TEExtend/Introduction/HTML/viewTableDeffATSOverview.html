Trouble shooting

If you see a blank page to the right, complete the following steps:
  • Check the file and folder to ensure that the folder is named 'ATSLab' and the file is named 'atsBaseView.xml', case is sensitive
  • Check that the folder has read and execute permission for all users
    • Do this by running the command 'chmod 755 /var/www/ATSLab' from a command line
  • Go back and try to save the XML table definition again
