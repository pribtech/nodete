Goal:

Create a menu folder for your tutorials.

Steps to complete:

  • Create a new folder in TE/tutorials called 'myTutorials'
  • Create a new file, menu_04_myTutorials.xml
  • Open menu_04_myTutorials.xml with a text editor
  • Make the changes as outlined in the file to the right by following these steps: (Refer to the legend below)
    • Set the menu type to "branch"
    • Set the branchDirectory to "myTutorials"
    • Set the description to "My Tutorials"

Notes:

  • The type 'branch' is used to designate menu directories and 'leaf' types are used to designate tutorials
  • Menus are generated using any xml files starting with menu and ending in an xml extension within the directory. The menu is sorted alphabetically using these xml files.
  • Only menus that contain actual tutorials will be added to the 'Learn' menu so you will not be able to see the new menu item yet.
