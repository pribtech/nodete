Goal:

Add sub-menu folders to your menu structure.

Steps to complete:


  • Create a new sub-menu folder, 'Personal'

    • Create a new folder in TE/tutorials/myTutorials called 'personalTutorials'
    • Create a new file, menu_01_personalTutorials.xml
    • Open menu_01_personalTutorials.xml with a text editor
    • Make the changes as outlined in the first file to the right labelled menu_01_personalTutorials.xml by following these steps: (Refer to the legend below)
      • Set the menu type to "branch"
      • Set the branchDirectory to "personalTutorials"
      • Set the description to "Personal Tutorials"
    • Save menu_01_personalTutorials.xml

  • Create a new sub-menu folder, 'Original'

    • Create a new folder in TE/tutorials/myTutorials called 'originalTutorials'
    • Create a new file, menu_02_originalTutorials.xml
    • Open menu_02_originalTutorials.xml with a text editor
    • Make the changes as outlined in the second file to the right labelled menu_02_originalTutorials.xml by following these steps: (Refer to the legend below)
      • Set the menu type to "branch"
      • Set the branchDirectory to "originalTutorials"
      • Set the description to "Original Tutorials"
    • Save menu_02_originalTutorials.xml
