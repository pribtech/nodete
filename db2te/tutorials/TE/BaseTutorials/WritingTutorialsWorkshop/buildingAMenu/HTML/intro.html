In this tutorial you will learn how to create an organizational menu structure for your tutorials by completing the following steps:

  • Create a new main menu folder
  • Add a tutorial to the new menu structure
  • Create sub menu folders
  • Add a set of tutorials to the new menu structure

You will build a menu structure around the tutorial created during the second tutorial of this workshop: 2. Using the Tutorial Template. If you have not already completed the tutorial, please do so before continuing.
