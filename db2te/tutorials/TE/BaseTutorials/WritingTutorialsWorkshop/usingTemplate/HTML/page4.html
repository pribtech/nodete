Goal:

Customize the tutorialScritp.xml file to create a unique tutorial title.

Steps to complete:

  • Open tutorialScript.xml with a text editor
  • Change the title of the tutorial. Make the changes as outlined in the file to the right by following these steps: (Refer to the legend below)
    • Change the value of the 'name' attribute of the 'tutorial' tag to 'New Tutorial From Template'
  • Save tutorialScript.xml

See the effects of your work

(Click here for an example of the expected results of this step)

  • If 'New Tutorial' is already open, then reload it using the reload icon to the left of the tutorial title
  • If 'New Tutorial' is not already open then navigate to Learn → Technology Explorer → New Tutorial


You should now see that the tutorial name has been changed in both the main tab title and the tutorial title next to the reload icon.


The changes are indicated with the following:


  • Indicated text has been removed
  • Indicated text has been edited
  • Indicated text has been added
  • Indicated text is the source of a copy
