Goal:

Customize the menu xml file to create a unique tutorial name.

Steps to complete:

  • Open menu_04_newTutorial.xml with a text editor
  • Change the name of the tutorial. Make the changes as outlined in the file to the right by following these steps: (Refer to the legend below)
    • Change the value of the 'description' tag to "New Tutorial"
    • Change the value of the 'parameter' tag with a name attribute 'tutorialName' to "newTutorial"
  • Save menu_04_newTutorial.xml

See the effects of your work

(Click here for an example of the expected results of this step)

  • Reload the menu by navigating to File → Reload Menu
  • Navigate to Learn → Technology Explorer → New Tutorial


You should now see that the tutorial name has been changed in the menu.


The changes are indicated with the following:


  • Indicated text has been removed
  • Indicated text has been edited
  • Indicated text has been added
  • Indicated text is the source of a copy
