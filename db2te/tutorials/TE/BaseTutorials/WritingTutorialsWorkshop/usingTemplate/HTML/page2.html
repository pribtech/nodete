Goal:

Create a copy of the Tutorial Template to use as a base for your own tutorial.

Steps to complete:

  • Create a copy of menu_03_template.xml located in TE/tutorials/TE/
  • Rename the copy as "menu_04_newTutorial.xml"
  • Create a copy of the tutorialTemplate folder located in TE/tutorials/TE/
  • Rename the copy as "newTutorial"

See the effects of your work

(Click here for an example of the expected results of this step)

  • Reload the menu by navigating to File → Reload Menu
  • Navigate to Learn → Technology Explorer


You should now see a copy of the Tutorial Template in the menu. Your new tutorial is the second tutorial titled 'Tutorial Template' since you named the xml file as 05_menu_newTutorial.xml.

The menu is generated using all xml files beginning with 'menu' contained within the directory in increasing alpha-numeric order.
