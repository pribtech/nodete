Check Your Privileges

Make sure that you have SYSADM (DB2 System Administrator) privileges on the DB2 instance that you are configuring this plug-in for.

Also, verify that you have DBADM (Database Administrator) privileges on the DB2 database that will contain the authentication information.

Steps:

  1. Connect to the database that you wish to store authentication data in, using the 'Connection Manager' tab.

  2. Switch to the 'Ad hoc SQL' tab, and click on 'Run.' Verify that there is at least one 'Y' in each row.
