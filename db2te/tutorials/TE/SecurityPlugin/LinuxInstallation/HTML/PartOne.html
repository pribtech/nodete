Check the Pre-requisites

Pre-requisites for Linux platforms:

  • On 64-bit Linux platforms, libssl and libcrypto (OpenSSL libraries) 0.9.8 or higher must be installed.
  • The ncurses library must be installed (e.g /lib/libncurses.so)



Note: These installation and set-up steps were written based on the README documentation that was originally supplied with the plug-in package.
