The Master View Tab

The Master View, like the Group Memberships tab, also displays all membership information. The query for this tab display is displayed in the 'Ad hoc SQL' tab to the right.

The 'Master View' tab next to the 'Ad hoc SQL' tab is the actual Master View tab that the module pulls up. If the tab has not loaded properly, you have not installed the plug-in, are not connected to the proper database, or do not have the proper privileges. In that case, click here for a sample view.

There are seven menu options (three each under the Add and Delete menus):

  • Add → User
  • Add → Group
  • Add → Membership
  • Delete → User
  • Delete → Group
  • Delete → Membership
  • Change Password

You will also see six action icons:

  •   (left) to remove the user
  •   to display more authorization details about the user (such as privileges and roles) in the bottom details panel (see page 7 of this tutorial for a picture)
  •   to change the user's password
  •   (center) to remove the membership
  •   to display more authorization details about the group
  •   (right) to the group

Steps:

  1. Click on the 'Master View' tab located to the right of the 'Ad hoc SQL' tab
  2. Try the different menus and icons

Note: If a user does not belong to any group, the group column for that user will appear as "null." Similarly, if a group has no users, the user column for that row will be "null."
