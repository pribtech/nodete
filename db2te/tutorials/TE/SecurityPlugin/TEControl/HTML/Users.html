The Users Tab

The Users tab displays all content in the DB2AUTH.USERS table except the password. The query for this tab display is displayed in the 'Ad hoc SQL' tab to the right.

The 'Users' tab next to the 'Ad hoc SQL' tab is the actual Users tab that the module pulls up. If the tab has not loaded properly, you have not installed the plug-in, are not connected to the proper database, or do not have the proper privileges. In that case, click here for a sample view.

There are three menu options:

  • Add User
  • Change Password
  • Delete User

You will also see three action icons:

  •   to remove the user
  •   to display more authorization details about the user (such as privileges and roles) in the bottom details panel (see page 7 of this tutorial for a picture)
  •   to change the user's password

Steps:

  1. Click on the 'Users' tab located to the right of the 'Ad hoc SQL' tab
  2. Try the different menus and icons
