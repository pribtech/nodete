The Group Memberships Tab

The Group Memberships tab displays all membership information. The query for this tab display is displayed in the 'Ad hoc SQL' tab to the right.

The 'Group Memberships' tab next to the 'Ad hoc SQL' tab is the actual Group Memberships tab that the module pulls up. If the tab has not loaded properly, you have not installed the plug-in, are not connected to the proper database, or do not have the proper privileges. In that case, click here for a sample view.

There are four menu options (two under the Edit Membership menu):

  • Add User
  • Add Group
  • Edit Membership → Add User to Group
  • Edit Membership → Delete User from Group

You will also see three action icons:

  •   to remove the membership
  •   to display more authorization details about the user (such as privileges and roles) in the bottom details panel (see page 7 of this tutorial for a picture)
  •   to display more authorization details about the group

Steps:

  1. Click on the 'Group Memberships' tab located to the right of the 'Ad hoc SQL' tab
  2. Try the different menus and icons

Note: If a user does not belong to any group, the group column for that user will appear as "null." Similarly, if a group has no users, the user column for that row will be "null."
