Troubleshooting

If you see a blank page to the right, complete the following steps:
  • Check the file and folder to ensure that the folder is named 'db2authLab' and the file is named 'db2authCustomMasterView.xml.' Case is sensitive.
  • Go back and try to save the XML table definition again.
After each step, reload this page by going back to the previous page, then returning to this page.
