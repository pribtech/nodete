Adding a Reference

Many of the pages in the Technology Explorer include references to related tables. For example, the Columns pages, the Indexes page, and the Table Privileges page all include a reference link to the Table page to display the table related to the column, index or privilege.

In this example, you will create a reference to the table defined in the Authority/authids.xml file.

Descriptions of edits being made:

  • We need to define what the column type ('reference') and names are of the new columns to display
  • The references need to be defined, just like other columns
    • Here, both are referencing the table, 'Authority/authids' which is an XML table definition file in [your_TE_directory]/tableDefinitions/Authority folder
    • In the first reference, we are matching 'userid' from our table definition to the 'authid' in 'Authority/authids'
    • In the second reference, we are matching 'groupid' from our table definition to the 'authid' in 'Authority/authids'
