How to export data from Ad Hoc SQL

This tutorial shows the steps that a user needs to take for exporting the data to XML, CSV or HTML format using the Ad Hoc SQL.

  1. First of all, write the Qd Hoc SQL in the window corresponding to the data that needs to be exported.
  2. Next, execute the statement.
  3. If the Ad Hoc SQL executes successfully and returns data, then at the header bar you will be seeing the controls for exporting the data in specific format (XML, CSV or HTML).
