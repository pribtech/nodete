IBM Data Movement Tool

Prior to DB2 9.7 for Linux, UNIX, and Windows Migration Tool Kit(MTK) used for migrating Oracle schema and applications to DB2. Beginning with DB2 V9.7 , the Migration Toolkit (MTK) is not required. IBM Data Movement Tool replaces the MTK functionality with a greatly simplified workflow. This article presents a very simple and powerful tool that enables applications from Oracle to be run on DB2 Version 9.7 for Linux, UNIX, and Windows.
