Create more orders to test packages

Now by using anonymous block and simple flow control statements, you will generate the random customers, products and random orders to simulate the application regressly.

Anonymous block in the right pane will generate

  • 10 random customers using create_customer procedure of customer_package package
  • 20 random products using create_product procedure of product_package package
  • 150 random orders using create_order procedure of order_package package


After running the anonymous block you can see several orders generated by the random customers.

CLPPlus

DB2 9.7 offers Command Line Processor Plus (CLPPlus) which provides a command-line user interface that you can use to connect databases and to define, edit, and run statements, scripts, and commands. You can watch a video in Enablement Videos section under Enablement Resources on the left pane on how to use CLPPlus.
