Create MENU in Technology Explorer

By now, we have our XML profile created and copied in respective directory. Now we need to create a menu item in Technology Explorer. Menu items in TE are also XML profiles.

Steps Involved:

  • Refer to the sample menu XML profile on right pane.
  • Open a text editor and copy the sample menu XML profile presented on the right pane
  • Modify the section highlighed in yellow according to the view XML profile name and its location.
    • Description element is the textual description of the view to be created.
    • If the file is saved as 'testTabDef.xml' in 'test' directory under tableDefinitions directory, then the parameterList element will look like this: 
<parameterList>
    <parameter name="table">test/testTabDef</parameter>
    <parameter name="action">list_table</parameter>
</parameterList>
  • Once the XML profile for menu is ready, save the file as menu_50_testTabDef.xml in the menu directory which can be found under the TE base directory. The number 50 in the file name identifies where the order of this menu item relative to the other menus in TE. testTabDef is any name you give for this menu.
  • Once the xml file is copied, click on "File-->Reload Menu". (Note: 'File' menu is part of Technology Explorer, not the browser's main menu)
  • Once the menu is reloaded, you will see a new menu item in main TE menu list. When you click on this menu, it will open the view created in a new pane.
  • You can find more examples of views under 'View, Statistics, Health and Tools' menu.
