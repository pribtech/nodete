Switch User for trusted connection


From previous pages, we saw Bob is using a trusted connection. Switch user from Bob to Sue to use trusted connection.

Operation


Double click on icon under "Set TC" column against Bob's trusted connection. It will pop-up a window to enter user name and password of the user to be switched to. So enter user name and password for user Sue and click on OK button.

Note: From trusted context object definition, Sue requires authentication.

Solution


Now, user is switched from Bob to Sue to use trusted connection. You will see a new entry in connection manager window with Sue as switched user(You will see Sue's name under "Set TC" column).
