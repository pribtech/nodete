Switch User for trusted connection


From previous pages, we saw Joe created a trusted connection. Switch user from Joe to Bob to use trusted connection.

Operation


Click on icon under "Set TC" column against Joe's trusted connection. It will pop-up a window to enter user name and password of the user to be switched to. So enter Bob and click on OK button.

Note: From trusted context object definition, Bob does not require authentication.

Solution


Now, user is switched from Joe to Bob to use trusted connection. You will see a new entry in connection manager window with Bob as switched user(You will see Bob's name under "Set TC" column).
