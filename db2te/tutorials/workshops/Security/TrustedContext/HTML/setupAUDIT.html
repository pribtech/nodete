The 'credit_card' table is accessed by different users for different purpose. SECADM Pat wants to audit the employees, managers, and administrators for any activities on that table.

Operation


Auditing can be enabled on different objects like tables, roles, trusted context objects, authorities (SYSADM, DBADM, SECADM) and database.

Pat would like to audit the trusted context object. Based on this requirement, he performs the following:

  • Create audit policies to:
    • Audit access to 'credit_card' through trusted connection.
    • Audit privileges of users accessing 'credit_card' through trusted connection.
  • Enable audit on trusted context object.
