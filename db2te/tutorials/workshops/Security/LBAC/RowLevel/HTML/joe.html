Access to table by manager Joe

You must log into the database as Joe before proceeding with this page.

Joe is the general manager of customer service department for coastal north and south regions. He is granted security label 'CENTRAL'.

Check which regions of data can Joe access from the 'dailysales' table.

Operation


Execute SELECT statement on the right pane.

Solution


Since Joe is granted with security label 'CENTRAL', he is able to access all record of 'dailysales' table which are associated with security labels NORTH and SOUTH.
