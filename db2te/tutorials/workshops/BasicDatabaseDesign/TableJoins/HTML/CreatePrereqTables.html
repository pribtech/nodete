Note : If the table is already created, it will indicate so on the console. Proceed to the step no. 3 in this case


Prerequisites

Make sure the 'product' table is created

Operation

  1. Click on the action button in the action panel
  2. It creates the 'product' table, if not already created


Result

The 'product' table is created and populated with data. If the table already exists, it will indicate so
