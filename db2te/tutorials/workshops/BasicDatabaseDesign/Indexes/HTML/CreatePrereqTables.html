Prerequisites

Make sure the tables enabled for 3NF with appropriate constraints applied, are created, as in the previous tutorial

Operation

  1. Click on the action button in the action panel
  2. It creates the store tables, if not already created, enabled for 3NF with constraints enforced


Result

The store tables with enabled for 3NF and constraints are created. If the tables already exist, it will indicate so
