Run the query in the Ad Hoc command window on your right.

As you can see in the Ad Hoc command window, we have to cast the data from the abstract column we inserted in the step before into a VARCHAR type. Without this, the data would be displayed as (CLOB) and could not be read. You can try changing the parameter of the VARCHAR to display more characters from the CLOB column. You can also remove the whole casting to see what looks like a CLOB column without it.
