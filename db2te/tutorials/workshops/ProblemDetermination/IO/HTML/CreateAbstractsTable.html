Run the query on your right to create the ABSTRACTS table.

To be able to produce an I/O saturation on your system, you need to create a table that will hold a certain amount of LOBs. The ABSTRACTS table will be used for this purpose in every steps of this tutorial.

Note that to create CLOB greater than 1GB, you must specify the NOT LOGGED option. More information on this option is available here: NOT LOGGED INITIALLY.

You should see a successful statement message in the console panel.
