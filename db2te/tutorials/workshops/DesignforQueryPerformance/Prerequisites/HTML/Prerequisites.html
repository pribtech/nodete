Prerequisites

This page describes the instance and database prerequisites required for the workshop.

If you are using a VM image provided by the IBM DB2 team, the environment may have already been set up for you. Check with the documentation that comes with the image and proceed to the next section as appropriate.

Note :
  1. Memory used by this workshop : ~1.5GB
  2. Database size (after creation of prerequisite tables) : ~ 2.5 GB
