Note : If the tables are already created, it will indicate so on the console. Proceed to the step no. 3 in this case


Prerequisites

Make sure a multi-partition processing instance and the designdb database are created before proceeding. Please refer to the Design Overview section for detailed instructions.

The prerequisite tables are created in order to enable creation of the replicated MQT

Operation

  1. Click on the action button in the Action panel
  2. It creates a replicated MQT table. The dimension table is replicated in the factspace containing the fact tables.
