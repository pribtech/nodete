Prerequisites

Make sure a multi-partition processing instance and the designdb database are created before proceeding. Please refer to the Design Overview section for detailed instructions.

The prerequisite tables are created in order to run the query against tables before and after compression and compare the size savings as well as query performance improvement

Operation

  1. Click on the action button on the Action panel
  2. It creates tables enabled for multidimensional clustering and populates them with data.
