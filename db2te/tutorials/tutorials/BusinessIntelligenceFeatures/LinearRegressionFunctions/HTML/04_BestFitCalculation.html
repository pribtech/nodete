The best fit is a value between 0 and 1 which tells you how well the equation y = ax + b (with the computed values a and b) fits the actual data supplied. A value of 1 means that we have a perfect fit, while a value of 0 means that there is no correlation).

When you use the random data, chances are that you will get a very low best fit, normally 0 or slightly better. Every time you regenerate the random data you will get different best fit values.
