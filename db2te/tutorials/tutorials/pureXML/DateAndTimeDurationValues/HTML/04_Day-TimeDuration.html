The data type xdt:dayTimeDuration represents a duration of time that is expressed using days, hours, minutes and seconds components. The range that can be represented by this data type is from -P11574074074DT1H46M39.999999S to P11574074074DT1H46M39.999999S (this is approximately 31,000,000 years).

  • nD, n represents the number of days.
  • T the date and time separator.
  • nH, n represents the number of hours.
  • nM, n represents the number of minutes.
  • nS, n is an unsigned decimal that represents the number of seconds. If a decimal point appears, it must be followed by one to six digits that represent fractional seconds.
