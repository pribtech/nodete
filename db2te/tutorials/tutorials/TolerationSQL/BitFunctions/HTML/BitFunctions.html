The BIT function

A new BIT function was introduced in DB2 9.5 to handle bit manipulation. There are 5 new functions:

  • BITAND
  • BITANDNOT
  • BITOR
  • BITXOR
  • BITNOT

Depending on the datatype, the number of bits can range from 16 to 113. The datatypes that can be used with these functions includes SMALLINT, INTEGER, BIGINT, and DECFLOAT. All other datatypes get converted to DECFLOAT.
