The transaction table specifies how the invoice table should be updated. There are three types of transactions:

  • insert (I) - all of the data in this record in the transaction table is inserted into the invoice table,
  • delete (D) - invoice id in this record in the transaction record is used to delete a record from the invoice table, and
  • update (U) - not null values in this record in the transaction table are used to update information in the invoice table.
