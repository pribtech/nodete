Use the CREATE EVENT MONITOR statement to create and define a monitor that will record statistics of events that occur when using the database.

These particular queries will also create the following wlm-target-tables with default table names:

  • ACTIVITYSTMT_DB2ACTIVITIES
  • ACTIVITYVALS_DB2ACTIVITIES
  • ACTIVITY_DB2ACTIVITIES
  • CONTROL_DB2ACTIVITIES
  • CONTROL_DB2STATISTICS
  • HISTOGRAMBIN_DB2STATISTICS
  • QSTATS_DB2STATISTICS
  • SCSTATS_DB2STATISTICS
  • WCSTATS_DB2STATISTICS
  • WLSTATS_DB2STATISTICS
