Create a service class and then create a workload such that all activities run from the WLM_tutorial_wk1 application get mapped to the newly created service class.

There are a number of attributes that can be specified when creating a workload or a service class. For example, when creating a workload, you can identify the connection based on application name, session user, etc. For more information, refer the CREATE WORKLOAD and the CREATE SERVICE CLASS documentation.
