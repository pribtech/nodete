From the previous output, we can tell which transaction handle (TranHdl) app2. We can now find which locks this transaction is waiting on by issuing the db2pd locks command for that transaction handle.

  1. Issue the command (replace Y with the trasaction handle #): 
    db2pd -db sample -locks Y wait
    • Look for the Type field and Owner field.
  2. Keep the CLP or terminal open
