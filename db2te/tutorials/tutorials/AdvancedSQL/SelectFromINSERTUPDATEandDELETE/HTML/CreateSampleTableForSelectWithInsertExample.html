This SQL will create the sample table that will be used to demonstrate the SELECT from I/U/D statement. Note that one column (EMP_Bonus) contains a generated column that is always calculated as EMP_Salary / 10 (10% of the salary). By using a SELECT statement as part of the INSERT, we can retrieve the bonus value that is generated by the system rather than having to re-SELECT the rows. In addition, there is a tax column that is set to 10 (percent). This value is the default for all rows unless it is explicitly entered.
